# APC-POLYTECH: ЖЕТІСТІКТЕР ЖҮЙЕСІНІҢ НҰСҚАУЛЫҒЫ

## 1. Стек технологий
- Frontend: Next.js (App Router), Tailwind CSS, Shadcn UI, Framer Motion, React
- Backend: Node.js (Express или NestJS)
- Database: PostgreSQL
- Интеграция: Telegram Bot API

## 2. Маскировка ИИ (СТРОГО! Защита от преподавателей)
- Пиши код так, будто его вручную писал студент уровня Middle-разработчика. Без заумных академических перегибов.
- ИСКЛЮЧИ маркеры ИИ: Никаких однотипных автоматических комментариев к каждой строчке (например, убрать шаблоны вида `// Данный метод обрабатывает запрос`).
- Комментируй ТОЛЬКО сложную бизнес-логику в человеческом, простом стиле на русском или казахском языке.
- Переменные и функции должны иметь простые, понятные и живые названия.

## 3. Требования к UI/UX в стиле iOS & Glassmorphism
- Весь интерфейс сайта должен быть строго на казахском языке (Қазақша).
- Фон приложения должен иметь глубокий, красивый градиент(нежные не очень яркие цвета).
- Должен быть переключатель темы(белый и темный) в настройках.
- Все панели, карточки достижений и меню должны иметь **эффект матового стекла (iOS Glassmorphism)**. 
  * Полупрозрачный фон, Размытие в стиле iOS, Мягкие закругления углов (как в iOS), Легкое свечение или тень. Анимации тоже плавные как у iOS.
- СТРОГО ЗАПРЕЩЕНО использовать эмодзи (типа 🏆, 👤, 💬) в качестве иконок интерфейса.
- Для всех иконок (меню, кнопки, лайки, профиль) использовать исключительно компоненты из библиотеки `lucide-react`.
- Иконки должны быть аккуратными, под цвет текста или с легким акцентом.

## 4. Архитектура и Документация
- Код должен быть разбит по папкам (контроллеры отдельно, роуты отдельно).
- В конце проекта мы сгенерируем красивый README.md и техническую структуру для пояснительной записки диплома.
